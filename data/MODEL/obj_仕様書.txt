// マテリアル名
mtllib ファイル名

・.mtlのファイル名が入っている。

// 頂点
v 〜

・xyzwの順に格納。ただし、wはない場合もある

// テクスチャ座標
vt 〜

・uvの順に格納。

// 法線
vn 〜

・xyzの順に格納。

// 面データ
f 〜

・４パターン有り
	└ 1 : f v0/t0/n0 v1/t1/n1 ...
		・頂点インデックス,テクスチャ座標インデックス,法線インデックスの順。
	
	└ 2 : f v0//n0 v1//n1 ...
		・頂点インデックス,法線インデックスの順( テクスチャ座標の『 / 』有り )
	
	└ 3 : f v0/n0 v1/n1 ...
		・頂点インデックス,法線インデックスの順( テクスチャ座標の『 / 』なし )
	
	└ 4 : f v0 v1 ...
		・頂点インデックスのみ。
		
・構成する要素によって、f以下の要素数が長くなる。よって、改行文字まで調査する必要有り。

// 面に使用する色
usemtl 〜

・使用するマテリアルの名前が格納されている。
