==libjpeg利用ガイド==

1:利用の際は、ライセンスを明記すること。
2:現在libにあるのは、ビルドツールセット『v100』で生成したもの。
	他のバージョンを利用する際は注意すること。
3:インクルードの際は、以下の通りに設定すること。

extern "C"
{
     #define XMD_H
     #include "jpeglib.h"
     #include "jerror.h"
}

#ifdef _DEBUG
#pragma comment( lib, "jpegd.lib" )
#else
#pragma comment( lib, "jpeg.lib" )
#endif
